# Projet Keaver

Ce repository contient le backend et le front end d'un petit projet que j'ai fait pour le CLV du Lycée chevrollier Angers. Ce projet avait pour but d'organiser une vente de fleurs avec un site.

## Front end 
#### Technologies / Librairies / Framework Utilisées

- [VueJs](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Axios](https://www.npmjs.com/package/axios)
- [Chart.js](https://www.npmjs.com/package/chart.js)
- [Vue Router](https://www.npmjs.com/package/vue-router)



## Backend
#### Technologies / Librairies / Framework Utilisées

- [Spring Boot](https://spring.io/)
- [Lombok](https://projectlombok.org/)
- [MongoDB](https://www.mongodb.com/)
- [JavaJWT](https://github.com/auth0/java-jwt)
